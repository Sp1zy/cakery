# Munamii Cakery Website

A professional, responsive multi-page web presence built for a boutique pastry shop transitioning from an Instagram-only model to an official storefront online.

## 🚀 Live Demo
You can view the live deployed website here: https://sp1zy.github.io/cakery/

## ✨ Features
- **Multi-Page Structure:** Consists of 4 distinct semantic pages: Home, About, Products, and Contact.
- **Consistent Navigation:** Identical header and footer layouts synced across all sub-pages for a seamless user experience.
- **Advanced Product Layout:** Uses a responsive CSS Grid system with `auto-fit` functionality to showcase 8+ cupcakes and 8+ wedding cakes. The layout beautifully transitions from a 4-column grid on desktop screens to a single-column layout on mobile devices.
- **Modern UI Elements:** Replaced pixelated background images with soft, clean CSS linear gradients mimicking professional studio lighting. Added card hover micro-interactions for a premium feel.

## 🛠️ Tech Stack
- HTML5 (Semantic Structure)
- CSS3 (Flexbox, CSS Grid, Custom Gradients, Transitions)

## 📌 Requirements Fulfilled
- Minimum of 4 unique pages.
- At least 8 item entries per product section.
- Working email links (`mailto:`) and verified profile redirects in the footer.
